#DragonKill

使用 C# 製作、文字 RPG 遊戲，玩家需要探索地圖、擊敗怪物，最終挑戰龍。

---
#遊戲玩法
玩家從「樹根」出發

可選擇不同路線（左上 / 右上 / 左下 / 右下）

每個區域都有敵人與獎勵

收集裝備（劍、法杖、龍心、鑰匙）

---

#戰鬥系統

三種攻擊方式：

拳頭（物理傷害）

劍（物理傷害）

法杖（魔法傷害）


爆擊系統：

所有攻擊行為有 30% 爆擊機率

傷害公式：傷害 = 攻擊力 - 敵人防禦

（爆擊先計算，再扣防禦）

---
#遊戲系統設計

- Game：控制流程
- BattleSystem：戰鬥邏輯
- StoryManager：劇情
- UiService：畫面顯示
- Inventory：背包系統
- Enemy：敵人資料
- LocationType (enum)：地圖位置

#遊戲攻略

風妖會飛，你不能用劍跟拳頭打他，除非他掉下來
